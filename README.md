## Artificial Intelligence & Machine Learning Notebooks

This repository contains a collection of **AI and ML notebooks** created for learning, experimentation, and hands-on practice. Each notebook demonstrates specific algorithms, model training workflows, or problem-solving techniques.

---

## Contents

### 1. **N-Queens Solvers**
- Implementations using **Constraint Satisfaction Problem (CSP)**, **A\*** search, and **Hill Climbing** algorithms.  
- Demonstrates **problem-solving, heuristic search, and algorithm design skills**.

### 2. **Machine Learning Models**
- Training notebooks for **classification** and **regression** tasks.  
- Includes **data preprocessing, feature engineering, scaling, model training**, and **evaluation** using multiple algorithms like **Logistic Regression & Random Forest**.  
- Some notebooks highlight **model validation and leakage detection**, illustrating **critical ML workflow practices**.

### 3. **Other AI Algorithms**
- Notebooks on **Search Algorithms** and other AI techniques.  
- Focused on building a **conceptual and practical understanding** of AI problem-solving strategies.

---

## Key Skills Demonstrated

- **Data preprocessing and feature engineering**  
- **Model training and evaluation** (classification & regression)  
- **Implementation of classical AI algorithms** (search, CSP, heuristics, RL)  
- **Ethical and responsible ML practices**, including **leakage detection** and dataset validation  
- Use of Python libraries: **pandas, numpy, scikit-learn, matplotlib**

## How to Use

Open the Jupyter/Colab notebooks in this repository and run the cells to reproduce results.
